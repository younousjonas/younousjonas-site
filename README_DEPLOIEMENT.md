# YounousJonas.com - site statique v0.1

Ce dossier contient une première version reconstruite de YounousJonas.com.

## Pages

- `index.html` : accueil
- `a-propos.html` : page auteur
- `livres.html` : bibliothèque réunionnaise
- `le-malokisme.html` : page narrative autour du livre
- `conferences.html` : conférences et ateliers
- `contact.html` : contact qualifié

## Configuration

Modifier `assets/config.js` pour ajuster :

- email
- URL LeMalokisme.com
- URL Komsa.re
- liens Amazon
- WhatsApp public si besoin
- webhook Make.com si besoin

## Déploiement Git

1. Dézipper le dossier.
2. Remplacer la zone portrait par une vraie photo si souhaité.
3. Modifier `assets/config.js`.
4. Pousser dans le dépôt Git.
5. Déployer sur Netlify, Vercel ou autre hébergement statique.
6. Pointer le domaine OVH vers l’hébergeur choisi.

## Rôle du site

YounousJonas.com installe la voix et la confiance. Il route vers :

- LeMalokisme.com pour la campagne dédiée au livre.
- Komsa.re pour les parcours, offres, inscriptions et accompagnements.
